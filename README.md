# Shiw Instruments
- Based on [redm-instruments]([https://github.com/mudcube/MIDI.js](https://github.com/kibook/redm-instruments))
- Create music in real-time with other players
- Uses [MIDI.js](https://github.com/mudcube/MIDI.js) to provide an array of instruments, configurable soundfonts and MIDI file playback
- Interactions with instrument props and different animations while idle vs. playing
- Added MIDI Sync for sounds (If u choose piano - it play on piano midi)
- Removed choose style of insrument (Like Electro Guitar) for immerssive playing
- Fixed async animation (piano fly)


# Commands

| /instrument [instrument]
| /instrument reset (if bugged)
